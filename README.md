# 203hotline_2018
2nd year mathematics computer science mini project written in C++.

USAGE

      ./203hotline [n k | d]
      
DESCRIPTION

      n n value for the computation of (nk)
      k k value for the computation of (nk)
      d average duration of calls (in seconds)
      
EXEMPLE
      
      ./203hotline 100 18
      
      ./203hotline 180
